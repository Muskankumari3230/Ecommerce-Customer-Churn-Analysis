# E-commerce Customer Churn Analysis

## Overview
This project analyzes customer churn behavior in an e-commerce platform using a Kaggle dataset. The analysis is conducted using Microsoft Excel pivot tables to identify factors influencing customer churn and retention.

## Dataset
- Source: Kaggle
- Type: E-commerce customer transactional data
- Total Records: 5,630 customers

## Tools Used
- Microsoft Excel (Pivot Tables)
- Microsoft Word (Project Report)
- GitHub (Project Hosting)

## Analysis Performed
- Churn overview analysis
- Customer satisfaction vs churn
- Order amount hike vs churn
- Coupon usage vs churn
- Order count vs churn
- Days since last order vs churn

## Key Insights
- Customer engagement is the strongest factor affecting churn
- Higher price hikes increase churn probability
- Coupon-dependent customers show higher churn
- Satisfaction alone does not guarantee retention
